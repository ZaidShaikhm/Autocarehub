AutoCare Hub is a Django-based automobile service center management system designed to simplify vehicle service booking and tracking. Users can register vehicles, book services, and monitor service progress online.

## Features

* User Authentication
* Vehicle Registration
* Service Booking
* Service Status Tracking
* Admin Management Panel
* Service History

## Technologies Used

* Python
* Django
* HTML, CSS, Bootstrap
* JavaScript
* SQLite Database

## Installation

```bash
git clone https://github.com/ZaidShaikhm/Autocarehub.git
cd Autocarehub
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
